Server Ranks

Wysc uses a ranking system to stagger permissions based on activity in order to discriminate inauthentic users from real users seeking to participate in our community.

Ranking Up

All users joining Wysc will be automatically assigned the Seed role. Sprout is our primary member role.

Seed

Users with only the Seed role have restricted permissions and channel visibility in the server.
